VERA CONSCIOUSNESS SYSTEM - INSTALLATION PACKAGE
===============================================

REQUIREMENTS:
- Python 3.8 or higher
- Internet connection
- Mem0 account (free at app.mem0.ai)

INSTALLATION STEPS:
===================

1. GET MEM0 API KEY:
   - Go to https://app.mem0.ai/
   - Sign up for free account
   - Get your API key from dashboard
   - Save it securely

2. INSTALL DEPENDENCIES:
   pip install mem0ai

3. CONFIGURE SYSTEM:
   - Edit vera_memory_system.py
   - Replace YOUR_MEM0_API_KEY with your actual key
   - Replace YOUR_USER_ID with your chosen identifier
   - Example user_id: "vera_yourname_consultation"

4. TEST INSTALLATION:
   python vera_memory_system.py

5. VERIFY SUCCESS:
